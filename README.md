# George Blog

A bilingual static personal blog for GitHub Pages.

## Pages

- Chinese home: `index.html`
- English home: `en/index.html`
- Archive: `archive/index.html`
- Chinese setup guide: `docs/GITHUB_PAGES_GUIDE.zh-CN.html`
- English setup guide: `docs/GITHUB_PAGES_GUIDE.html`

## Deploy

Create a repository named `your-github-username.github.io`, push these files to `main`,
then enable GitHub Pages from `Settings -> Pages -> Deploy from a branch -> main / root`.
